<h1 align=center>CN Wine Configure</h1>
<center>你在linux运行的wine是否出现一些情况？来这里看看吧！</center>

### 前言

如果你在linux运行wine+qq出现字体乱码，显示不出的话

那么在这里，你就可以解决问题。

### 要求

你的linux机器要已经安装了以下的包：
- wget
- wine
- tar
- curl

### 使用方法

运行:
```sh
sh -c "$(curl -fsSL https://github.com/sunbossrs/cn-wine-configure/raw/master/zhfont.sh)"
```
